RISE MCP Server v1.1.0

rise-mcp — Model Context Protocol server for RISE CRM. Connect your RISE instance with AI assistants like Claude, Cursor, Windsurf, GitHub Copilot, Continue.dev, and OpenCode.

Installation

npm install -g rise-mcp

Or run directly without installing:

npx rise-mcp

Requires Node.js >= 20 and a RISE CRM instance with the Rise API module activated.

Configuration

Set these environment variables before starting the server:

VariableDescriptionExample
RISE_API_URL Base URL of your RISE CRM instance https://tu-empresa.risecrm.com
RISE_API_KEY API Key generated in RISE CRM sk-abc123...

You can also create a .env file in the working directory with these variables.

Usage with AI Assistants

To use the MCP server with your AI assistant, configure it in your assistant's MCP settings:

Claude Desktop / Cursor / Windsurf

{
  "mcpServers": {
    "rise-mcp": {
      "command": "npx",
      "args": ["-y", "rise-mcp"],
      "env": {
        "RISE_API_URL": "https://tu-empresa.risecrm.com",
        "RISE_API_KEY": "sk-abc123..."
      }
    }
  }
}

OpenCode

Add to your project's .opencode/mcp.json:

{
  "mcpServers": {
    "rise-mcp": {
      "command": "npx",
      "args": ["-y", "rise-mcp"],
      "environment": {
        "RISE_API_URL": "https://tu-empresa.risecrm.com",
        "RISE_API_KEY": "sk-abc123..."
      }
    }
  }
}
