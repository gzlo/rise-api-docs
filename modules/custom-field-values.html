Custom Field Values 4 endpoints

Controlador de valores de campos personalizados (custom field values).

Endpoints

MethodPathDescription
GET /api/v1/custom-field-values Lista valores de custom fields, filtrado por entidad y registro.
POST /api/v1/custom-field-values Crea o actualiza (upsert) el valor de un custom field en un registro.
GET /api/v1/custom-field-values/bulk Asigna múltiples custom fields a un registro en una sola llamada. Las keys de `fields` pueden ser: - ID numerico: { "2": "USD", "5": "NIF-12345" } - Titulo: { "Currency": "USD", "NIF": "NIF-12345" } Body: { "related_to_type": "invoices", "related_to_id": 5, "fields": { "2": "USD", "5": "NIF-12345" } }
DELETE /api/v1/custom-field-values/{id} Elimina (soft delete) el valor de un custom field.

GET /api/v1/custom-field-values

Lista valores de custom fields, filtrado por entidad y registro.

Required Scopes

custom_fields:read

Query Parameters

NameTypeDescription
page string Filter by page
limit string Filter by limit
related_to_type string Filter by related to type
related_to_id string Filter by related to id
custom_field_id string Filter by custom field id

Example Request

curl -X GET https://your-rise.com/api/v1/custom-field-values \
  -H "Authorization: Bearer rise_your_token"

Example Response

{"success": true, "data": {...}}

Client restricted Staff and admin only. Clients cannot access this endpoint.

POST /api/v1/custom-field-values

Crea o actualiza (upsert) el valor de un custom field en un registro.

Required Scopes

custom_fields:write

Request Body Parameters

NameTypeRequiredDescription
related_to_id int Required Integer value
custom_field_id int Required Integer value
value string Required Text value

Example Request

curl -X POST https://your-rise.com/api/v1/custom-field-values \
  -H "Authorization: Bearer rise_your_token" \
  -H "Content-Type: application/json" \
  -d '{
    "related_to_id": 1,
    "custom_field_id": 1,
    "value": "string_value"
}'

Example Response

{"success": true, "message": "Created", "data": {"id": 1}}
