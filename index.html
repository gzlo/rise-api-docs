Getting Started

1

Generate an API Key

Go to Settings → API Keys in your RISE installation and generate a new token. Choose the appropriate scopes for your use case.

2

Authenticate your requests

Include your token in the Authorization header as a Bearer token:

curl -H "Authorization: Bearer rise_your_token_here" \
  https://your-rise.com/api/v1/me
3

Make your first API call

Try creating a task or listing your projects. All responses follow the same format:

{"success": true, "data": {...}}
{"success": false, "error": {"code": "...", "message": "..."}}
